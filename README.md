# ASL3 AutoLink (Auto-Reconnect)

A tiny systemd + script helper for AllStarLink / ASL3 that:
- Auto-connects your node to a hub/other node on boot
- Auto-reconnects if the link drops

## Install (recommended)

Clone and run the installer:

```bash
git clone https://github.com/<YOURNAME>/asl3-autolink.git
cd asl3-autolink
sudo bash install.sh
